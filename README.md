## 编辑和贡献小贴士 / Edit & Contribute Tips

- 记得先安装字体
- 请使用 Inkscape 编辑 svg 文件
- 如果发现 Inkscape 导出的 png 是黑白色的... 请点击导出按钮右边的小齿轮看看设置
- 提交 PR 前，请确保 png 文件都是最新的 (记得导出)
